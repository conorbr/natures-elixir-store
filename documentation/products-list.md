## Next Steps

### Product Details Needed

For each product, we need to determine:

1. **Variants/Options**

   - Tea: Size options (50g, 100g, 250g, etc.)
   - Essential Oils: Volume options (10ml, 30ml, 50ml, etc.)
   - Supplements: Size/weight options
   - Soaps: Size options (if applicable)
   - Sponges: Size options (Small, Medium, Large)

2. **Pricing**

   - EUR prices (in cents)
   - GBP prices (in pence)
   - Price per variant

3. **SKU Codes**

   - Unique SKU for each variant
   - Format: `CATEGORY-PRODUCT-VARIANT`

4. **Descriptions**

   - Product descriptions
   - Health benefits (if applicable)
   - Usage instructions

5. **Images**

   - Product images
   - To be uploaded via admin dashboard

6. **Shipping Profiles**
   - Default profile: Tea, Sponges, Supplements, Soaps
   - Fragile profile: Essential Oils (glass containers)
